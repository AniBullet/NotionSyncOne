# 更新日志

## v1.1.0 - 排版优化版本（2025-12-01）

### ✨ 新功能

#### 1. 📸 正文图片自动上传
- **自动上传**：同步时自动提取并上传所有图片到微信素材库
- **URL 替换**：自动将原始图片 URL 替换为微信服务器 URL
- **批量处理**：支持一次上传多张图片，带进度日志
- **永久可访问**：图片保存在微信素材库，不会失效

#### 2. 💻 代码块美化
- **深色主题**：使用深色渐变背景，提升代码可读性
- **语言标签**：自动显示编程语言名称
- **主题适配**：代码块配色随主题变化
- **语法标识**：支持 Notion 中设置的所有语言

#### 3. 🔗 超链接优化
- **简洁展示**：只显示链接文本，不再冗余显示完整 URL
- **主题配色**：链接颜色随主题自动调整
- **下划线效果**：使用主题色下划线，更加美观
- **点击友好**：保持完整的跳转功能

#### 4. 🎨 六大主题样式系统
- **default（默认）**：简洁大方，蓝色系，适合技术文章
- **wechat（微信）**：微信绿，商务风格，贴近官方
- **hongfei（红绯）**：红色系，热情洋溢，醒目活泼
- **jianhei（简黑）**：黑色系，酷炫个性，极简主义
- **shanchui（山吹）**：橙黄系，温暖明亮，亲和友善
- **chengxin（橙心）**：橙色系，活力四射，年轻动感

### 🔧 技术改进

- **批量上传**：优化图片上传流程，支持并发处理
- **错误处理**：上传失败自动回退到原始 URL
- **日志增强**：详细的上传进度和状态日志
- **性能优化**：减少重复上传，提高同步速度

### 📚 文档更新

- 新增 `STYLING_GUIDE.md` - 详细的使用指南
- 新增 `CHANGELOG.md` - 版本更新日志
- 更新配置说明文档

### 🐛 Bug 修复

- 修复图片 URL 特殊字符转义问题
- 优化代码块在微信编辑器中的兼容性
- 改进主题样式的内联 CSS 处理
