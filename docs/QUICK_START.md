# 🚀 快速开始 - 排版优化功能

## 一分钟配置主题

### 1. 打开配置页面
在应用左侧菜单点击"配置"

### 2. 选择主题
在微信公众号配置区域，添加或修改主题设置：

```json
{
  "theme": "wechat"
}
```

可选主题：
- `default` - 默认蓝色主题
- `wechat` - 微信绿色主题（推荐）
- `hongfei` - 红色主题
- `jianhei` - 黑色极简主题
- `shanchui` - 橙黄色温暖主题
- `chengxin` - 橙色活力主题

### 3. 保存并同步
点击保存 → 选择文章 → 点击同步

## 常见问题

### Q: 图片上传失败？
**A:** 检查以下几点：
1. 服务器 IP 是否在微信白名单中
2. 网络连接是否正常
3. 图片大小是否 < 5MB
4. 查看详细日志信息

### Q: 主题不生效？
**A:** 
1. 确认配置已保存
2. 重新同步文章
3. 清除缓存后重试

### Q: 代码块样式被修改？
**A:**
1. 微信编辑器可能调整部分样式
2. 建议发布前在编辑器中预览
3. 可以在编辑器中微调

### Q: 想要自定义主题？
**A:**
1. 查看 `src/shared/types/theme.ts`
2. 添加新主题配置
3. 在配置中使用新主题名

## 进阶技巧

### 1. 组合使用
```json
{
  "theme": "wechat",
  "author": "张三",
  "topNotice": "本文来自 Notion 知识库\n欢迎关注订阅"
}
```

### 2. 不同类型文章使用不同主题
- 技术文章 → `default` 或 `jianhei`
- 企业公告 → `wechat`
- 创意内容 → `hongfei` 或 `chengxin`
- 温馨分享 → `shanchui`

## 推荐工作流

1. **准备阶段**
   - 在 Notion 中编写文章
   - 添加高质量封面图
   - 代码块指定编程语言

2. **配置阶段**
   - 选择合适的主题
   - 设置顶部提示语
   - 检查作者配置

3. **同步阶段**
   - 同步文章到微信
   - 查看日志确认成功
   - 特别注意图片上传状态

4. **发布阶段**
   - 在微信后台预览
   - 微调细节（如需要）
   - 发布或定时发布
